# Python Lernrepo

Sammlung von Python-Lerninhalten und Übungsaufgaben.

## Struktur

```
Python/
├── Lerninhalt/   → Erklärungen & Referenz pro Thema
└── Übung/        → Aufgaben als Mini-Projekte
```

## Lerninhalt

| Datei | Thema |
|-------|-------|
| `datentypen.py` | int, float, str, bool, None, Typumwandlung |
| `strings.py` | Methoden, Slicing, f-Strings, split/join |
| `funktionen.py` | def, return, *args, **kwargs, lambda, scope |
| `operatoren.py` | Arithmetik, Vergleich, logisch, bitweise |
| `listen.py` | Methoden, Slicing, Schleifen, verschachtelt |
| `tuples.py` | Unveränderlich, Unpacking, Konvertierung |
| `sets.py` | Mengenoperationen, frozenset |
| `dictionaries.py` | Zugriff, Schleifen, verschachtelte Dicts |
| `string_formatierung.py` | %-Operator, .format(), f-Strings, Hex-Farben |
| `regex.py` | re.match, search, findall, split, sub, E-Mail-Prüfung |

## Übung

| Datei | Aufgabe |
|-------|---------|
| `aufgabe_datentypen.py` | Persönliches Profil — Typumwandlung & Ausgabe |
| `aufgabe_strings.py` | Textanalyse — Wörter zählen, suchen, ersetzen |
| `aufgabe_operatoren.py` | Taschenrechner & Berechtigungssystem |
| `aufgabe_funktionen.py` | Notenrechner — def, return, *args |
| `aufgabe_filme.py` | Filmsammlung — Tuples & Listen kombiniert |
| `speisekarte.py` | Restaurant — Funktionen, Tuples, Listen |
| `aufgabe_sets.py` | Kursteilnehmer — Mengenoperationen |
| `aufgabe_dictionaries.py` | Kontaktbuch — verschachtelte Dicts |
| `log_analyst.py` | Log-Datei-Analyst — Regex, enumerate, 3 Formatierungsarten |
