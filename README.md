# Python Lernrepo

Sammlung von Python-Lerninhalten, Uebungsaufgaben und kleinen Mini-Projekten.

## Struktur

Dieses Repository enthaelt aktuell zwei Ordnungsstile:

- `Lerninhalt/` und `Übung/` fuer die urspruengliche Sammlung
- `Lerninhalte/` und `Uebungen/` fuer die neu hochgeladenen Inhalte

## Lerninhalte

Die Lernordner enthalten Grundlagen und Referenzen zu Themen wie:

- Datentypen
- Strings
- Funktionen
- Operatoren
- Listen
- Tuples
- Sets
- Dictionaries

Zusaetzlich gibt es in der urspruenglichen Sammlung weitere Themen wie:

- `string_formatierung.py`
- `regex.py`
- `datum_zeiten.py`
- `methoden_iterables.py`

## Uebungen und Projekte

Enthalten sind klassische Uebungsdateien wie:

- `aufgabe_datentypen.py`
- `aufgabe_strings.py`
- `aufgabe_operatoren.py`
- `aufgabe_funktionen.py`
- `aufgabe_filme.py`
- `aufgabe_sets.py`
- `aufgabe_dictionaries.py`
- `speisekarte.py`
- `f1_analyse.py`

Ausserdem liegen mehrere kleine Projekte im Repository:

- `Uebungen/smart_library.py`
- `Uebungen/tiktaktoe.py`
- `Uebungen/SmartCarHouse/`
- `Uebungen/smarthomesystem.py/`
- `Uebungen/smarthomesystem/`
- `Uebungen/bibApi/src/bibApi/`
- `Uebungen/Galaktische_Bücherei.py/`

## Hinweis

Einige Inhalte liegen aktuell doppelt bzw. in leicht unterschiedlicher Ordnerstruktur vor. Das README beschreibt deshalb bewusst den aktuellen Stand der Sammlung statt nur einer einzigen Zielstruktur.
