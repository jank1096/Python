# Python Lernrepo

Eine Sammlung meiner Python-Lerninhalte, Uebungsaufgaben und kleinen Mini-Projekte.

Dieses Repository dokumentiert meinen Weg durch Python-Grundlagen, Datenstrukturen, Funktionen, OOP und erste strukturierte Projektansaetze.

## Struktur

```text
Python/
+-- Lerninhalte/   Grundlagen und Referenzen pro Thema
+-- Uebungen/      Aufgaben, Experimente und Mini-Projekte
```

## Lerninhalte

Im Ordner `Lerninhalte/` liegen kompakte Beispiele und Notizen zu wichtigen Python-Basics:

| Datei | Thema |
| --- | --- |
| `datentypen.py` | Zahlen, Strings, Booleans, None und Typumwandlung |
| `strings.py` | String-Methoden, Slicing, f-Strings und Textverarbeitung |
| `operatoren.py` | Arithmetik, Vergleiche und logische Operatoren |
| `listen.py` | Listen, Methoden, Schleifen und verschachtelte Daten |
| `tuples.py` | Unveraenderliche Sequenzen und Unpacking |
| `sets.py` | Mengen, Schnittmengen, Vereinigungen und Differenzen |
| `dictionaries.py` | Key-Value-Strukturen, Zugriff und verschachtelte Daten |
| `funktionen.py` | Funktionen, Parameter, Rueckgabewerte und Scope |

## Uebungen

Der Ordner `Uebungen/` enthaelt Aufgaben zu den Grundlagen und kleine Anwendungen:

| Datei | Inhalt |
| --- | --- |
| `aufgabe_datentypen.py` | Einstieg in Datentypen und Ausgaben |
| `aufgabe_strings.py` | Textanalyse und String-Operationen |
| `aufgabe_operatoren.py` | Rechnen, Vergleiche und Bedingungen |
| `aufgabe_funktionen.py` | Funktionen und einfache Logik |
| `aufgabe_filme.py` | Listen und Tuples anhand einer Filmsammlung |
| `aufgabe_sets.py` | Mengenoperationen mit Teilnehmerdaten |
| `aufgabe_dictionaries.py` | Arbeiten mit Dictionaries |
| `speisekarte.py` | Kleine Restaurant-Logik mit Funktionen |
| `f1_analyse.py` | Einfache Analyse von Formel-1-Daten |
| `smart_library.py` | Kleine Bibliotheksverwaltung |
| `tiktaktoe.py` | Konsolenbasiertes Tic-Tac-Toe |

## Mini-Projekte

Einige Uebungen sind als kleine Projektordner aufgebaut:

| Ordner | Beschreibung |
| --- | --- |
| `Uebungen/SmartCarHouse/` | OOP-Beispiele rund um Fahrzeuge |
| `Uebungen/smarthomesystem.py/` | Erste Smart-Home-Variante mit einzelnen Klassen |
| `Uebungen/smarthomesystem/` | Strukturierteres Smart-Home-Projekt mit Package-Aufbau |
| `Uebungen/bibApi/src/bibApi/` | Kleine Bibliotheks-API mit Medien, Buechern und Magazinen |

## Themen, die im Repo geuebt werden

- Python-Grundlagen und Syntax
- Datentypen und Datenstrukturen
- Funktionen und einfache Programmlogik
- Objektorientierte Programmierung
- Kleine Konsolenprogramme
- Package-Struktur und erste API-Ansatze

## Ziel

Das Repo dient als persoenliche Lernsammlung. Es soll zeigen, welche Python-Konzepte ich bereits geuebt habe und wie sich einzelne Aufgaben schrittweise zu kleinen Projekten entwickeln.
