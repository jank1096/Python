# Python – Fachinformatiker Lernprojekte

Dieses Repository enthält alle Python-Übungen und Projekte aus meiner Ausbildung zum Fachinformatiker.

## Setup

- Python 3.13
- Editor: VS Code mit Black Formatter, Ruff, Pylance

## Struktur

```
Python/
├── grundlagen/       # Variablen, Datentypen, Kontrollstrukturen
├── funktionen/       # Funktionen, Parameter, Rückgabewerte
├── datenstrukturen/  # Listen, Dictionaries, Tupel, Sets
├── oop/              # Klassen, Objekte, Vererbung

```

## Ausführen
