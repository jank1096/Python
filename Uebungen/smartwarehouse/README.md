# Smart Warehouse

Kleines Python-OOP-Projekt fuer eine Lagerverwaltung.

## Inhalte

- Produktklassen mit Vererbung
- Eigene Exceptions
- Warehouse-Klasse zum Verwalten, Verkaufen und Filtern
- History mit Zeitstempel
- JSON-Export
- Einfaches Terminal-Menue in `main.py`

## Struktur

```text
smartwarehouse/
├── main.py
├── pyproject.toml
├── README.md
└── src/
    └── smartwarehouse/
        ├── __init__.py
        ├── exceptions.py
        ├── models.py
        └── warehouse.py
```

## Start

```bash
cd Uebungen/smartwarehouse
python3 main.py
```
